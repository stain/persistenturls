OCLC PURL 2 Project: Requirements

To view HTML pages for the requirements:
  1.  Run ./build.sh
  2.  Open ../site/project/purl/documentation/requirements/index.html

To edit the source:
  1.  Copy the file template.xml into a new filename
  2.  Uncomment tags as necessary.  Tag names are (hopefully) explanatory.  If not, ask Dave (david@zepheira.com).
