If you're seeing this page via a web browser, it means you've successfully Setup NetKernel with your Servlet Engine. Congratulations!

If you are the administrator you may access the backend fulcrum for administration and documentation.

Note: If you see Exceptions when accessing the backend system it is because your servlet engine does not have the XML libraries (found in lib/endorsed/ below the NetKernel install directory) in its endorsed classpath. For more info boot in standalone mode and look at this guide.
